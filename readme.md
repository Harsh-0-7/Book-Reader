# Book Reader

![Demo Image](Demo.jpeg)

Main feature of this reader is to use read aloud feature for free.

Right now only pdf format is supported.

## How to Use?

- Open pdf.html
- Upload PDF
- Click on Read Aloud Button (Need Internet)
